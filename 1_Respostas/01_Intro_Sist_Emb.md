### O que são sistemas embarcados?
São sistemas encarregados de um conjunto de funções pré-determinados para resolver um problema específico de forma eficiente.


### O que são sistemas microprocesssados?
É um dispositivo composto por uma Unidade Central de Processamento(CPU) e um conjunto de periféricos.


### Apresente aplicações de sistemas embarcados: (a) Para a indústria automotiva; (b) Para eletrodomésticos; (c) Para automação industrial.
A) Controle do Freio ABS, Painel de controle
B) Geladeiras, microondas, Internet das Coisas. 
C) Construção de carros, Mondagem de placas de circuito impresso e integrado.

### Cite arquiteturas possíveis e as diferenças entre elas.
 - FPGA: É um hardware programável;

 - Microcontroladores Eles possuem uma baixa capacidade de processamento e servem principalmente para controle de periféricos;

 - Microprocessadores: Possuem uma alta capacidade de processamento, comparada com microprocessadores, onde cabem SOs;

 - Asics: São os SoC (System on Chip/Circuito Integrado) é um hardware não programável, são altamente otimizados para uma tarefa específica.
